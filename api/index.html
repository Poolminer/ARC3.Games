getTopSequences gameId

level		 order ("ASC" or "DESC", default "ASC")

limit (maximum number of results, default no limit)		 Get the top sequences on record of a game's level, ordered by action count.

A sequence is represented as an array of action ids, except for clicks, which are represented as an array with the x and y value respectively.

Sequences are guaranteed to finish the level, and not to contain resets, unavailable actions, or out of bounds clicks. 		curl -X POST https://arc3.games/api.php \
-H "Content-Type: application/x-www-form-urlencoded" \
-d 'action=getTopSequences&gameId=ls20-9607627b&level=1&order=ASC&limit=3'
